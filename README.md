Sistema de Cadastro de Reservas

O projeto atual é um protótipo para criação de um aplicativo de cadastro de resevas de quartos para alunos de uma faculdade. 
O aplicativo deve possuir no escopo a hora da reserva, o tempo esperado, o nome do aluno, matrícula, data, perído e verificação de cadastro.
